# Zazu Adventures Frontend

React + Vite application for the Zazu Adventures travel website.

## Overview

This app powers the public-facing website and includes sections such as:

- homepage hero content
- testimonials sourced from WordPress
- reusable UI sections and utilities

## Stack

- React 19
- Vite
- Tailwind CSS
- React Router
- Lucide icons

## Getting Started

From the `zazu/` directory:

```bash
npm install
npm run dev
```

## Available Scripts

- `npm run dev` - start the development server
- `npm run build` - build the app for production
- `npm run preview` - preview the production build
- `npm run lint` - run ESLint across the codebase

## Content Source

Some homepage content is loaded from the WordPress API, including testimonials.
Keep that data shape in mind when changing the UI or service layer.
